# Hướng dẫn thêm QR Code thanh toán

Vui lòng thêm file ảnh QR Code chuyển khoản ngân hàng với tên `bank.jpg` vào thư mục này.

## Yêu cầu:

- Tên file: `bank.jpg`
- Kích thước khuyến nghị: 500x500 pixels
- Định dạng: JPG/JPEG
- Nội dung: Mã QR chuyển khoản ngân hàng của bạn

## Cách tạo QR Code:

1. Truy cập ứng dụng ngân hàng của bạn
2. Chọn "Nhận tiền" hoặc "Tạo mã QR"
3. Tải ảnh QR Code về
4. Đổi tên thành `bank.jpg`
5. Copy vào thư mục `public/images/`

Sau khi thêm file, QR Code sẽ tự động hiển thị trong trang thanh toán.
